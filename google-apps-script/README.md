# Google Sheets: сохранение данных эксперимента

## 1. Откройте вашу таблицу
В меню: **Расширения → Apps Script**.

## 2. Вставьте код
Удалите шаблон `Code.gs` и вставьте содержимое файла `Code.gs` из этой папки. Сохраните.

## 3. Разверните веб-приложение
1. **Развернуть → Новое развёртывание**
2. Тип: **Веб-приложение**
3. Выполнять от имени: **Меня**
4. У кого есть доступ: **Все**
5. Нажмите **Развернуть**, подтвердите права
6. Скопируйте **URL веб-приложения**

## 4. Подключите к сайту
В корне репозитория откройте `config.js` и вставьте URL:

```js
window.EXPERIMENT_CONFIG = {
  googleSheetsUrl: 'https://script.google.com/macros/s/XXXX/exec',
};
```

Закоммитьте и запушьте изменение.

## 5. Проверка
Откройте URL веб-приложения в браузере — должно показать `ok: true`.
Пройдите эксперимент до конца: в листе `data` появится строка со всеми полями.

Лист `data` и заголовки создаются автоматически при первой записи.
